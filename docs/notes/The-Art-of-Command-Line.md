# The Art of Command Line

[Github Link](https://github.com/jlevy/the-art-of-command-line)

## Bash

| Keys       | What to do           | Addition |
| :--------- | :------------------- | :------- |
| `Ctrl+W`   | 删除 最后 一个单词   |          |
| `Ctrl + U` | 删除 光标 之前的内容 |          |
| `Ctrl + K` | 删除 光标 之后的内容 |          |
| `Alt + B`  | 向前 移动 光标       |          |
| `Alt + F`  | 向后 移动 光标       |          |
| `Ctrl + E` | 移动 光标 到开头     |          |
| `Ctrl + A` | 移动 光标 到结尾     |          |
| `Ctrl + I` | 清屏                 |          |
